## Getting Started 

Install dependencies,

```bash
$ yarn install
```

Compile rescript,

```bash
$ yarn rescript
```

Build documentation,

```bash
$ yarn dumi
```

Publish documentation via Github Pages,

```bash
$ yarn dumi:publish
```

## Interactive Playground

```bash
$ yarn install && yarn playroom
```
